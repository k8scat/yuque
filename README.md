# Yuque

Yuque Client SDK in Rust.

## Support

- [x] User
- [x] Group
- [x] Repo
- [x] Doc

## Usage

Add this to your `Cargo.toml`:

```toml
[dependencies]
yuque = "0.1.3"
```

## Unit tests

```bash
make test TOKEN=your_token
```
